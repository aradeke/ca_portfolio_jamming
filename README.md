# CA_Portfolio_Jamming

Dieses Projekt ist eine React-Anwendung, die als Portfolio dient. Es zeigt verschiedene Projekte an und bietet eine detaillierte Ansicht jedes Projekts.

## Installation

Um das Projekt lokal auszuführen, folgen Sie diesen Schritten:

1. Klonen Sie das Repository:
   ```
   git clone <repository-url>
   ```

2. Navigieren Sie in das Projektverzeichnis:
   ```
   cd CA_Portfolio_Jamming
   ```

3. Installieren Sie die Abhängigkeiten:
   ```
   npm install
   ```

## Starten der Anwendung

Um die Anwendung im Entwicklungsmodus zu starten, verwenden Sie den folgenden Befehl:
```
npm start
```
Die Anwendung wird standardmäßig unter `http://localhost:3000` ausgeführt.

## Struktur des Projekts

- **src/**: Enthält den Quellcode der Anwendung.
  - **components/**: Wiederverwendbare Komponenten wie Header, Footer und PortfolioCard.
  - **pages/**: Seitenkomponenten für die Anwendung.
  - **hooks/**: Benutzerdefinierte Hooks.
  - **styles/**: Globale CSS-Stile.
  - **utils/**: Hilfsfunktionen für API-Anfragen.
  - **types/**: Typen und Schnittstellen für TypeScript.
